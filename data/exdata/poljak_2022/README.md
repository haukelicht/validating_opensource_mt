The file `attack_data.xlsx` was kindly shared by Željko Poljak via email on November 4, 2023.

It records speeches from all _Question Time_ sessions in the sample, indicating which speeches contained attacks at other politicians/parties/governments.
Please note that it also includes protocol speeches, which are best to be removed (variable: protocol).

The main variables are:

- attack_binary: indicating if a politician attacked another politician/party/government in the speech.
- attack_total: the total number of attacks made in the speech.
- policy/trait/incivility: whether these categories are present in an attack. This is only coded if an attack is present.

Please **cite** his paper "Parties' attack behaviour in parliaments: Who attacks whom and when" (https://doi.org/10.1111/1475-6765.12542) when using the data.
